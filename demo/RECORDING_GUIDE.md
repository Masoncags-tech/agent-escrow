## Tools

**Recording:** 
- QuickTime (Mac) - Simple, free
- OBS - More control
- Loom - Easy sharing

**Editing:**
- CapCut (free, easy text overlays)
- DaVinci Resolve (free, pro features)

**Text Overlays:**
- Big, readable font (36pt+)
- High contrast (white on dark)
- Keep on screen 3+ seconds

---

## Music (Optional)
- Lo-fi beats (royalty free)
- Keep volume LOW (10-15%)
- No lyrics

---

## Export Settings
- 1080p or 1920x1080
- 30fps minimum
- MP4 format
- < 100MB for Twitter

3. Scroll through the code
4. Run the tests

Raw and authentic works for hackathons.
